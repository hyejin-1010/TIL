# 스케쥴링 알고리즘 기본
#### 프로세스란?
* 메모리에 올려져서, 실행 중인 프로그램
	* 코드 이미지(바이너리): 실행 파일. Ex) ELF Format
* 응용 프로그램 != 프로세스
	* 하나의 응용 프로그램은 여러 개의 프로세스(프로그램)가 상호작용하면서 실행될 수 있다.
	* 여러 프로그램을 만들어서, 서로 통신하면서 플호그램을 작성할 수 있다.(IPC 기법)

#### 스케쥴러
* 프로세스의 실행을 관리

## 최단 작업 우선(SJF) 스케쥴러
